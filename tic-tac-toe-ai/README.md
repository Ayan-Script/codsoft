# Tic-Tac-Toe AI

A browser-based Tic-Tac-Toe game with a polished frontend and an unbeatable Minimax AI opponent.

## Files

- `index.html` contains the app structure
- `styles.css` defines the responsive visual design
- `script.js` handles gameplay, UI updates, and the Minimax AI
- `tictactoe.py` starts a tiny local server for the frontend

## Run

```bash
python tictactoe.py
```

Then open `http://127.0.0.1:8000` in your browser.

## Features

- Responsive browser UI for desktop and mobile
- Symbol and first-turn controls
- Live status messaging and round scoreboard
- Minimax with Alpha-Beta Pruning for perfect AI play
